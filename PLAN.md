# Job Skill Intelligence --> sistema para estudiantes. 

## Flujo
1. Scraping → 
2. Limpieza → 
3. Extracción de skills →
4. Clasificación por área →  
5. Agregación → 
6. Visualización (dashboard)

---

## Ideas

This workflow could execute once a week. 

paso 1 --> conseguir -- nombre del puesto, empresa, requisitos, descripción -- de las ofertas de trabajo publicadas en Castellón de la Plana que tengan como requisito tener estudios universitarios. 

paso 2 --> de cada oferta extraer tipo de puesto, ámbito, habilidades y conocimientos prácticos. 

paso 3 --> presentarlo bonito.
